Atom编辑器

发表于 2015-11-25

基础操作

中文 英文 快捷键 备注
命令行 command line cmd+shift+p
项目跳转面板 delete line cmd+ctrl+p
文件跳转面板 Fuzzy Finder cmd+p
变量、函数跳转面板 Symbol-view cmd+r
语法选择 Grammar Selector ctrl+Shift+l

编辑

中文 英文 快捷键 备注
复制行 Duplicate Line ctrl-d(自定义)
删除行 Delete Line cmd-delete(自定义:未设置成功)
小写转换 lower-case cmd+k+l
大写转换 upper-case cmd+k+u
选择下一个 find-and-replace:select-next ctrl-g(自定义) 
自定义部分快捷键是为了统一android studio的操作.你也可以改为你熟悉的按键

插件

名称 说明 备注
timecop 启动时间管理
Save Session 保存会话
sublime-style-column-selection 块操作
minimap 代码地图
project-manager 项目列表/切换
filetype-color 图标美化
file-icons 同上
url-encode 对选中的文本进行编码解码
Sync Settings 保存你的atom的配置到gits
Pretty JSON 格式化json文件 有个不方便的地方,你需要先选择文件的语言为json
#快捷键
