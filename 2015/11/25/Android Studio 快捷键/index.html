Android Studio 快捷键

基础操作

名称 action 快捷键 备注
上下移动当前行 move line up/down alt+shift+up/down
删除当前行 delete line cmd+backspace
多文本选择 add selection for next occurrence Ctrl+G
列选择 Alt+MouseDrag
扩大/收缩选中部分 alt+up/down
合并行和文字 join lines ctrl+shift+j
代码提示 basic alt+/
当前文件方法列表 cmd+F12
查看方法调用路径 ctrl+alt+h

跳转

名称 action 快捷键 备注
最近打开的文件 cmd+e
最近编辑的文件 cmd+shift+e
显示使用的地方 alt+F7 cmd+alt+F7
跳到声明 cmd+b
跳到实现 cmd+alt+b
跳到类型声明 cmd+shift+b
跳到父类 cmd+u

Studio

名称 action 快捷键 备注
格式化代码 cmd+alt+l
包裹代码块 cmd+alt+t
移动方法 Move Statement cmd+delete+up/down
包裹代码块 Surround With cmd+alt+t 添加包裹代码块(if、for、try等)
反向包裹代码块 Surround With cmd+shift+delete 删除包裹的代码块
选择于 alt+F1 select Target,可用于快速打开文件目录
弹出VCS操作 Ctrl+V
查找方法 cmd+shift+a
隐藏所有面板 cmd+shift+F12
显示参数信息 cmd+p

书签

名称 action 快捷键 备注
开关书签 F3
用助记符开关书签 alt+f3
显示书签 cmd+F3
